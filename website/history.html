关键决策 / Key Decisions

决策 / Decision 方案 / Choice 理由 / Rationale
编译语言 / Language Python 标准库 / Python stdlib 零依赖，即开即用 / Zero dependencies
模块结构 / Structure core/ 包拆分 / Package split 单文件 1000+ 行难以维护 / Monolith too large
指令系统 / Directives @directive 装饰器 + 自动发现 新增指令只需创建一个 .py 文件
组件加载 / Components 约定式 + component.json 简单场景直接放文件，复杂场景显式声明
CSS 模板 / Themes @template 宏内联读取 / Inline macro 无需额外构建步骤 / No extra build step
注释语法 / Comments ! 前缀 与 @ 指令区分，避免混淆 / Distinguish from @ directives

已知问题 / Known Issues

  • Markdown 图片语法 ![alt](src) 在重构中丢失 — 可用 img[src=url][alt=text] 替代 / Was lost in refactoring — use img[src=url][alt=text] instead
  • attribute 值含空格时已修复引号嵌套问题 / Quote nesting fixed for attribute values with spaces
  • 尚无 CI/CD（GitHub Actions）/ No CI/CD yet
  • 尚无自动化测试框架 / No automated test framework yet
