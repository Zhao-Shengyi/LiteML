#sec1

1 | / Basic Structure

/ DOCTYPE

! html5 HTML5 Write ! html5 on the first line to declare HTML5.

 | ! html5 ← <!DOCTYPE html>

/ Comments

! @ Prefix with ! for comments (@ also works).

 | ! / This is a comment
 | ← <!-- / This is a comment -->

/ Indentation Nesting

2 Use 2-space indentation for parent-child — closing tags are automatic.

 | ! html5
 | html
 | head
 | title / Page Title
 | body
 | h1 / Hello
 | p / Body text

/ Void Elements

These tags self-close automatically:

meta input img br hr link area base col embed source track wbr 

 | meta[charset=UTF-8] ← <meta charset="UTF-8">
 | input[type=text][name=email] ← <input type="text" name="email">
#sec2

2 | / Tag Attributes

CSS / CSS Selector Syntax

tag#id.class1.class2[attr=val] / text

 | div#app.container.active[data-x=1] / content
 | ← <div id="app" class="container active" data-x="1"> / content</div>

.class #id / Shorthand

div Omitting the tag name defaults to div.

 | .container ← <div class="container">
 | #main.content ← <div id="main" class="content">

<head> @template(name) Apply a theme in <head> with @template(name).

 | head

#sec5

5 | / Layout Directives (LiteCSS)

@flex

 | div@flex(center, between)
 | p / Left
 | p / Right
 | ← <div style="display:flex;align-items:center;justify-content:space-between">

@grid

 | div@grid(cols=3, gap=16px)
 | .card 1 / Card 1
 | .card 2 / Card 2
 | .card 3 / Card 3

@waterfall / @position / @transition / @hide

— Masonry, positioning, transitions, responsive hiding — all in one line.

7 | / Behaviors

[*behavior] Add form validation with [*behavior] syntax.

 | input[type=text][*required][*email][*minlength=3]

/ Supports: *required *email *url *minlength *maxlength *range *pattern
