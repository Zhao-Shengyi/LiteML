1 基础结构 / Basic Structure

文档声明 / DOCTYPE

使用 ! html5 声明 HTML5 文档类型，仅第一行生效。 Write ! html5 on the first line to declare HTML5.

          ! html5  ←  <!DOCTYPE html>

注释 / Comments

使用 ! 前缀写注释（@ 前缀也兼容）。 Prefix with ! for comments (@ also works).

          ! 这是一条注释 / This is a comment
          ←  <!-- 这是一条注释 / This is a comment -->

缩进嵌套 / Indentation Nesting

2 空格缩进表示父子关系，编译器自动生成闭合标签。 Use 2-space indentation for parent-child — closing tags are automatic.

          ! html5
          html
            head
              title 页面标题 / Page Title
            body
              h1 你好 / Hello
              p 正文 / Body text

自闭合标签 / Void Elements

以下标签自动自闭合：These tags self-close automatically:

meta input img br hr link area base col embed source track wbr 

          meta[charset=UTF-8]  ←  <meta charset="UTF-8">
          input[type=text][name=email]  ←  <input type="text" name="email">

2 标签属性 / Tag Attributes

CSS 选择器语法 / CSS Selector Syntax

tag#id.class1.class2[attr=val] 文本 / text

          div#app.container.active[data-x=1] 内容 / content
          ←  <div id="app" class="container active" data-x="1">内容 / content</div>

.class 和 #id 简写 / Shorthand

省略标签名时默认 div。 Omitting the tag name defaults to div.

          .container    ←  <div class="container">
          #main.content  ←  <div id="main" class="content">

3 CSS 模板 / Templates

在 <head> 中用 @template(name) 应用主题。 Apply a theme in <head> with @template(name).

          head
            @template(minimal)
            title 我的页面 / My Page

内置 12 款模板：12 built-in themes: minimal, modern, dark, glass, neumorph, retro, corporate, blog, landing, card, gradient, terminal.

4 组件引用 / Components

使用 use 组件名 引入组件（@use 也兼容）。 Use use component-name to load a component (@use also works).

          use dark-mode-toggle
          ←  注入组件的 HTML + CSS + JS / Injects component HTML + CSS + JS

5 布局指令 / Layout Directives (LiteCSS)

@flex

          div@flex(center, between)
            p 左侧 / Left
            p 右侧 / Right
          ←  <div style="display:flex;align-items:center;justify-content:space-between">

@grid

          div@grid(cols=3, gap=16px)
            .card 卡片 1 / Card 1
            .card 卡片 2 / Card 2
            .card 卡片 3 / Card 3

@waterfall / @position / @transition / @hide

瀑布流、定位、过渡、响应式隐藏 — 全部一行指令搞定。 Masonry, positioning, transitions, responsive hiding — all in one line.

6 交互指令 / Interaction Directives (LiteJS)

无代码交互组件，自动注入 JavaScript。 No-code interactions with auto-injected JavaScript.

          button@modal(id=myModal) 打开弹窗 / Open Modal
          button@theme-toggle 切换主题 / Toggle Theme
          button@like 点赞 / Like

7 行为属性 / Behaviors

用 [*behavior] 语法添加表单验证。 Add form validation with [*behavior] syntax.

          input[type=text][*required][*email][*minlength=3]

支持 / Supports: *required *email *url *minlength *maxlength *range *pattern

8 PHP 模板引擎 / PHP Template Engine

使用 --mode=php 编译。 Compile with --mode=php.

变量输出 / Variable Output

          p {{ $user->name }}  ←  转义输出 / Escaped
          p {!! $rawHtml !!}  ←  原始输出 / Raw

控制流 / Control Flow

          @if($loggedIn)
            p 欢迎回来！/ Welcome back!
          @else
            p 请登录 / Please login
          @endif

循环 / Loops

          @foreach($items as $item)
            .card
              h3 {{ $item->title }}
          @endforeach

9 多媒体宏 / Media Macros 

          @audio(song.mp3 | controls)
          @video(video.mp4 | controls, width=100%)
          @embed(https://youtu.be/xxx | ratio=16:9)
          @icon(star)
          @css(path/to/style.css)
          @js(path/to/script.js | defer)

10 其他特性 / Other Features

长文本块 / Long Text Blocks

用 """ 包裹多段文字，自动拆分为 <p>。 Wrap with """ for auto-paragraph splitting.

          """
          第一段。空行分割段落。
          First paragraph. Blank line = new paragraph.
          
          第二段文字。
          Second paragraph.
          """

代码块 / Code Blocks

          ```python
          def hello():
              print("Hello!")
          ```

@raw 块 / Raw Blocks

原样输出，不做任何编译。 Output verbatim without compilation.

@eject 回退 / Eject Fallback

指令后加 ! 回退为原生实现。 Append ! to a directive to eject to native code.
