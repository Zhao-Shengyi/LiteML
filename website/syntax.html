语法参考 / Syntax Reference

完整语法手册，每个规则都配有 LiteML 源码与编译后 HTML 的对照。中英双语。

Complete syntax reference with LiteML source and compiled HTML side by side. Bilingual.

1. 基础结构 / Basic Structure

文档声明 / DOCTYPE

使用 ! html5 声明 HTML5 文档类型。仅第一行生效。

          | ! html5
          | ← <!DOCTYPE html>

注释 / Comments

使用 ! 前缀写注释（@ 前缀也兼容）。

          | ! 这是一条注释 / This is a comment
          | ← <!-- 这是一条注释 / This is a comment -->

缩进嵌套 / Indentation Nesting

2 空格缩进表示父子关系，编译器自动生成闭合标签。这是 LiteML 最核心的规则。

          | ! html5
          | html
          |   head
          |     title 页面标题
          |   body
          |     h1 你好
          |     p 正文内容

自闭合标签 / Void Elements

以下标签自动自闭合，无需手动添加 /：

meta input img br hr link area base col embed source track wbr

          | meta[charset=UTF-8]
          | input[type=text][name=email]
          | ← <meta charset="UTF-8">
          | ← <input type="text" name="email">

2. 标签属性 / Tag Attributes

CSS 选择器语法

一行语法：tag#id.class1.class2[attr=val] 文本内容

          | div#app.container.active[data-x=1] 内容
          | ← <div id="app" class="container active" data-x="1">内容</div>

.class 和 #id 简写

省略标签名时默认 div：

          | .container
          | #main.content
          | ← <div class="container">
          | ← <div id="main" class="content">

3. CSS 模板 / Templates

在 <head> 中用 @template(name) 应用主题。

          | head
          |   @template(minimal)
          |   title 我的页面

内置 12 款模板：minimal, modern, dark, glass, neumorph, retro, corporate, blog, landing, card, gradient, terminal。

4. 组件引用 / Components

使用 use 组件名 引入组件（@use 也兼容）。

          | use dark-mode-toggle
          | ← 注入组件的 HTML + CSS + JS

组件存放在 components/ 目录，支持约定式（template.html + style.css + script.js）和清单式（component.json）。

5. 布局指令 / Layout Directives (LiteCSS)

通过 @ 后缀在标签行附加布局指令。

@flex

          | div@flex(center, between)
          |   p 左侧
          |   p 右侧
          | ← <div style="display:flex;align-items:center;justify-content:space-between">

@grid

          | div@grid(cols=3, gap=16px)
          |   .card 卡片 1
          |   .card 卡片 2
          |   .card 卡片 3

@waterfall / @position / @transition / @hide

瀑布流布局、定位、过渡动画、响应式隐藏 — 全部一行指令搞定。

6. 交互指令 / Interaction Directives (LiteJS)

无代码交互组件，自动注入 JavaScript。

          | button@modal(id=myModal) 打开弹窗
          | button@theme-toggle 切换主题
          | button@like 点赞

7. 行为属性 / Behaviors

用 [*behavior] 语法添加表单验证行为。

          | input[type=text][*required][*email][*minlength=3]
          | ← 编译为带验证属性的 input + 自动注入 JS 验证逻辑

支持：*required *email *url *minlength *maxlength *range *pattern

8. PHP 模板引擎 / PHP Templates

使用 --mode=php 编译，支持 Blade 风格的控制流。

变量输出

          | p {{ $user->name }}
          | p {!! $rawHtml !!}
          | ← <p><?php echo htmlspecialchars( $user->name ); ?></p>

控制流

          | @if($loggedIn)
          |   p 欢迎回来！
          | @else
          |   p 请登录
          | @endif

循环

          | @foreach($items as $item)
          |   .card
          |     h3 {{ $item->title }}
          | @endforeach

@include

引入其他文件：@include(header.lite)

9. 多媒体宏 / Media Macros

          | @audio(song.mp3 | controls)
          | @video(video.mp4 | controls, width=100%)
          | @embed(https://youtu.be/xxx | ratio=16:9)
          | @icon(star)
          | @css(path/to/style.css)
          | @js(path/to/script.js | defer)

10. 其他特性 / Other Features

长文本块

用 """ 包裹多段文字，自动拆分为 <p>：

          | """
          | 第一段文字。空行分割段落。
          |
          | 第二段文字。
          | """

代码块

用 ``` 包裹代码：

          | ```python
          | def hello():
          |     print("Hello!")
          | ```

@raw 块

原样输出原始内容，不做任何编译：

          | @raw
          | @flex 这行不会被编译
          |   @if 也不会被处理
          | @endraw

@eject 回退

指令后加 ! 回退为原生实现：

          | div@flex(center, between)!
          | ← 展开为完整的 flex CSS，而非简写指令
